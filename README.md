# EMGesture - International IMS Student Contest

## TestWiFi
Test della connessione TCP del modulo WiFiNINA, calcolo velocità media in Kbps.

## letturaIMU
Lettura dati di accelerometro e giroscopio del sensore LSM6DSOX.
